## v2.0.1

> Adds the ChatGPT channel: proxy the chatgpt.com consumer web backend with a browser session cookie — OpenAI-compatible chat, thinking / pro / deep research (streamed chain-of-thought + report), web search, and image generation/edit — with a session-mode switch (normal / temporary / project) and automatic credential refresh.

### English

#### Added

- **ChatGPT channel (`chatgpt`).** Proxies the chatgpt.com consumer web backend via a browser **session cookie** (no API key / OAuth). Inbound pivots on OpenAI Chat Completions; the channel translates the SSE-v1 stream back into OpenAI chunks and surfaces reasoning as `reasoning_content`. Anti-bot state (sentinel proof-of-work + Cloudflare `__cf_bm` + Edge client-hints) is carried in the sealed secret. Native-only.
- **Thinking / pro / deep research over the conduit.** Thinking / pro / o-series turns and deep research stream their chain-of-thought and final report over the per-user conduit WebSocket. Consumer deep research is decoded from `update-widget-state` plan frames (live progress) plus the embedded report message.
- **Web search and image generation / edit.** Web-search citation markers are stripped from the answer; image gen/edit run as a channel-driven multi-step exchange.
- **Session mode (normal / temporary / project).** A per-provider setting controls where conversations land: normal persistent, temporary (privacy, the default), or inside a ChatGPT **project** — auto-created/found by name (default `gproxy`) so they stay grouped in your history. Surfaced in the console as a three-way selector.
- **Cookie-based credential auto-refresh.** The ~10-day access token is re-minted from the stored session cookie as it nears expiry, so the credential lives as long as the browser session — no manual re-paste until the cookie itself lapses.

### 简体中文

#### 新增

- **ChatGPT 渠道(`chatgpt`)。** 用浏览器**会话 cookie** 代理 chatgpt.com 消费版 web 后端(不是 API key / OAuth)。入站以 OpenAI Chat Completions 为枢轴;渠道把 SSE-v1 流翻回 OpenAI chunk,并把思维链以 `reasoning_content` 暴露。反爬状态(sentinel proof-of-work + Cloudflare `__cf_bm` + Edge client-hints)随密封 secret 携带。仅原生。
- **thinking / pro / 深度研究走 conduit。** thinking / pro / o 系列与深度研究的思维链和最终报告经 per-user conduit WebSocket 流式输出。消费版深度研究从 `update-widget-state` 计划帧(实时进度)+ 内嵌报告消息解码。
- **网页搜索与画图 / 改图。** 网页搜索的引用标记会从答案中剥离;画图/改图走渠道驱动的多步交换。
- **会话模式(普通 / 临时 / 进项目)。** 一个 per-provider 设置控制会话落点:普通持久、临时(隐私,默认)、或进 ChatGPT **项目** —— 按名自动建/找(默认 `gproxy`),方便在历史里分组查看。console 里呈现为三选一选择器。
- **基于 cookie 的凭据自动续期。** 约 10 天的 access token 在临近过期时用存着的会话 cookie 重新换取,凭据寿命跟着浏览器会话走 —— cookie 本身失效前无需手动重粘。
